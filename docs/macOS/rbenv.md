# rbenv

- https://github.com/rbenv/rbenv

```shell
brew install rbenv
# setup ur shell to load rbenv
rbenv init

# list latest stable version
rbenv install -l

# list all local versions
rbenv install -L

# install a Ruby version
rbenv install 3.2.2

# set the Ruby version for this machine
rbenv global 3.2.2

# set the Ruby version for this dir
rbenv local 3.2.2
```
